Project Vision

ShareShake Rental Web App started with ideas and values in share economy. We have defined our clients and their view to be those people interested in peer-to-peer based sharing of access to goods and services, Specifically we are interested in providing a platform that where clients can rent items that are needed only temporarily and at the same time they can also rent out unused items to other clients to earn money. On out platform, users can make posting for the items they want to rent out, and they can also search and find items they want to rent from other users.

Project Requirement

To explain a bit of the project requirement:

It is a web application accessible from anywhere with internet connection, that is responsible for providing the platform to rent and borrow products.

Users must be able to make payments for items using PayPal

Administrators must be able to manage and access user's account information

Owners must be able to see traffic information relating to the web application to help business decisions

Users must be able to create accounts to use the web application to be able to lend or borrow items

Project Plan

Kyung Neung Lee, Haitham AhmedKhalil, Faheem Choudhury and Karthik Sankar Prasad formed the Team 17 for ShareShake project. Our project is to provide rental specific web application platform started on September 15, 2017 and planned to complete by March 30, 2018. Which is an expansion into an area of the rental specific market which has not seen any developments and has an increasing demand. Once completed, the platform will allow users rent any categories of items without having to purchase the product for the full price.

Analysis and Design

We defiend renter and rentier as general group of users to be interacting with our web application.

The application should allow users to :

- Make account and edit information.

- Create, view, and search postings,

- Make transaction for the wanted items.
